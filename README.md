# Expense Tracker (Python CLI Project)

## Overview

The Expense Tracker is a simple command-line based Python application designed to help students manage and track their daily expenses. This project solves the common problem of unorganized spending by allowing users to record, view, and analyze their expenses efficiently.

---

## Problem Statement

Many students struggle to keep track of their daily expenses, which often leads to overspending and poor financial management. There is a need for a simple and accessible system to record and analyze expenses.

---

## Solution

This project provides a lightweight and user-friendly expense tracking system that allows users to:

* Record daily expenses
* Categorize spending
* View all transactions
* Analyze total and category-wise expenses

---

## Features

*  Add new expenses
*  View all recorded expenses
*  Calculate total spending
*  Category-wise expense summary
*  Persistent data storage using JSON

---

##  Tech Stack

* **Language:** Python
* **Libraries Used:**

  * `json` (for data storage)
  * `os` (file handling)
  * `datetime` (timestamping expenses)
